# 🎲 BoardGames Finder

## 📌 Descripción del proyecto

Hoy en día existe una gran variedad de juegos de mesa en el mercado y cada vez más personas se interesan por este tipo de ocio. La popularidad de los juegos de mesa ha crecido considerablemente en los últimos años, lo que ha provocado que los usuarios se encuentren con un catálogo muy amplio y difícil de explorar.

Muchas veces los jugadores no compran o no se atreven a probar juegos nuevos por falta de información, por no saber cuál se adapta mejor a cada situación o porque necesitan la recomendación de otra persona para decidirse.

Para solucionar este problema se ha desarrollado **BoardGames Finder**, una aplicación web que permite buscar, filtrar y descubrir juegos de mesa de forma sencilla, ayudando al usuario a encontrar el juego perfecto para cada momento.

La aplicación obtiene información desde una API externa y permite interactuar con los datos sin necesidad de recargar la página, ofreciendo una experiencia dinámica y fácil de usar.

---

## 🎯 Objetivo

Desarrollar una aplicación web interactiva utilizando **HTML, CSS y JavaScript**, aplicando los siguientes conceptos:

* Programación Orientada a Objetos (POO)
* Manipulación dinámica del DOM
* Consumo de APIs mediante fetch
* Uso de localStorage para guardar datos
* Creación de una interfaz interactiva y responsive

---

---

## 🌐 API utilizada

La aplicación utiliza una API externa de juegos de mesa para obtener información sobre los juegos, como:

* Nombre
* Imagen
* Número de jugadores
* Duración de la partida
* Categoría
* Descripción

Las llamadas a la API se realizan mediante `fetch` y se procesan de forma asíncrona.

## 🚀 Instalación y ejecución

1. Clonar el repositorio

```
git clone <URL_DEL_REPOSITORIO>
```

2. Entrar en la carpeta del proyecto

```
cd boardgames-app
```

3. Abrir el archivo index.html en el navegador

```

```

No es necesario instalar dependencias adicionales.
